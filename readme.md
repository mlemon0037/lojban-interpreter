## Usage

# Numbers

Numbers are specified in lojban format, see https://en.wikibooks.org/wiki/Lojban/Numbers
To make larger numbers, string together numbers
Ex. 35 = cimu

# Operators

Operators are specified in lojban
\- : vu'u
\+ : su'i
\* : pi'i
/ : fe'i

# Exiting the program

To exit the calculator, type dicra
